La comprobación no conecta
  1. Comprueba que has entrado dentro de una partida, no solo en la portada de Roll20.
  2. Confirma que Tampermonkey y ADOM External Sheet - Roll20 Bridge están activados.
  3. Usa el mismo navegador y el mismo perfil para la ficha y Roll20.
  4. Recarga tanto esta página como la pestaña de Roll20 después de instalar el puente.
  5. En Chrome o Edge, revisa que Tampermonkey tenga permiso para ejecutar scripts de usuario.
